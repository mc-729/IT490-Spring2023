API service

1. cd /etc/systemd/systemd
2. sudo touch rabbitMQAPI.service
3. sudo vim rabbitMQAPI.service
4. Copy and paste the contents of rabbitMQAPI.service file to the systemd file you just made
5. systemctl daemon-reload
6. sudo systemctl start rabbitMQAPI.service
7. sudo systemctl status rabbitMQAPI.service
